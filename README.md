# Expense Tracker - Трекер расходов

**Автор:** Стрыгин Ярослав

## Описание

Десктопное приложение для отслеживания личных финансов. Позволяет добавлять, редактировать и удалять расходы, фильтровать их по категориям и датам, а также подсчитывать общую сумму расходов за выбранный период.

## Функционал

### Основные возможности
- **Добавление расходов** с указанием суммы, категории и даты
- **Редактирование** существующих расходов
- **Удаление** расходов
- **Фильтрация** по категории
- **Фильтрация** по диапазону дат
- **Подсчёт суммы** расходов за выбранный период
- **Быстрые периоды**: сегодня, неделя, месяц, всё время
- **Сохранение и загрузка** данных в JSON-файл

### Валидация ввода
- Сумма должна быть положительным числом
- Дата должна быть в формате ГГГГ-ММ-ДД
- Категория выбирается из предустановленного списка

## Установка и запуск

### Требования
- Python 3.7 или выше
- Tkinter (встроен в стандартную установку Python)
